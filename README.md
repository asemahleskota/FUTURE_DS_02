 Telco Customer Churn Analysis Dashboard

 Objective
Analyze customer churn patterns and identify factors influencing customer retention using Power BI.

 Tools Used
- Power BI
- Power Query
- DAX

 Dataset
Telco Customer Churn Dataset

Key Insights
- Month-to-month contract customers show the highest churn.
- Long-term contract customers are more likely to stay.
- Electronic check is the most commonly used payment method.
- Average monthly charge is 64.80.

 Recommendations
- Encourage customers to switch to long-term contracts.
- Introduce loyalty rewards and retention campaigns.
- Monitor high-risk customer segments.
